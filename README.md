# 🛒 ShopSmart AI

An AI-powered Shopping Assistant built using Streamlit and Gemini AI.

## Features

- 🔍 Product Search
- 🌐 Website Scraper
- 💰 Discount Calculator
- 🤖 AI Review Summarizer

## Tools Used

- Streamlit
- Gemini API
- BeautifulSoup
- DuckDuckGo Search
- Requests

## Dynamic Variables

- Current Date
- User Query
- Budget
- Preferred Brand

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
